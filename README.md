# secured-app
# Защищенная версия приложения

### Инструкция по сборке и запуску приложения

> - git clone "текущий репозиторий"
> - cd app
> - pip install -r requirements.txt
> - python __init__.py
> - flask приложение будет запущено по адресу http://127.0.0.1:8089
